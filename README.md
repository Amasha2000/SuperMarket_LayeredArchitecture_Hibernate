# Super Market using Layered Architecture and Hibernate

This is an application for a Super Market using Layered Architecture and ORM Concepts.

Main Functions are,
 - Manage Customers
 - Manage Items
 - Place Orders
 - Manage Orders
 - Generate Bill
 - View Reports
 
 And also it has validated input fields.
 
 # Built With
 
 - Java
 - Java FX
 - Hibernate
 - MySql
 - Jasper Reports
 - And a little bit of CSS
